
<h1 align="center">
 Currency-Converter-App
</h1>

A Currency converter app that uses Fixer.io Endpoint to get the latest currency rates. Build using latest practices of React native.
Includes Redux-saga, React Navigation v5



## 🚀 Quick start

1.  **Clone Project**

    ```
    git clone https://github.com/usmanhaider123/currency-converter.git
    ```

2.  **Install node modules**

    Install the required node modules by running the command

    ```
    npm install or npm i
    ```
3.  **Run on Android**

    To run the App on Android device simply run

    ```
    npx react-native run-android
    ```
    
4.  **Run on iOS**

    To run the App on iOS device simply run

    ```
    npx react-native run-ios
    ```
